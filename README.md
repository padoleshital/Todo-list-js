
# 📝 To-Do App

A simple and intuitive To-Do list web application that allows users to add, delete, and mark tasks as complete.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Local storage support

## 🔧 Tech Stack
- HTML
- CSS
- JavaScript
